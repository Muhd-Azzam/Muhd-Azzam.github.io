Project 1 - Student Record Database

Student Record Database

This is a simple Student Record System (GUI) that was built using the Swing library in Java. It allows the users to manage and interact with a list of student records.

The users can add and remove student record into the system.After adding the student records into the system, the user can then sort the list of student based on their Name or Faculty
the users can press the save button to save the list into a save_file. after saving the users can load back the data by using the load button.
This program allow the user to record information and save it into a local database by saving the Array into a Save file

Check out code in Github repositories: https://github.com/Muhd-Azzam/StudentRecordDatabase

How do i use this?

It actually pretty easy!

  • Enter the student name and matrix number.
  • Press "Add" and it that student will be added to the records
  • Press "View Record" then it will list down all the record u made!
  • Press "Save" to save your records and "Load" to load back the records you made!

Technologies used

This program was built using multiple libraries in Java

  • Swing Library
  • Object Serialization
  • Java I/O (Input/Output)
  • Java GUI Components
